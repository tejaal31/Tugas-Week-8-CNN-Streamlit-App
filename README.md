# Tugas-Week-8-CNN-Streamlit-App
Proyek ini merupakan implementasi model Convolutional Neural Network (CNN) yang telah dioptimasi menggunakan teknik seperti hyperparameter tuning, dropout, dan transfer learning. Model diintegrasikan ke dalam aplikasi web sederhana berbasis Streamlit yang mampu menerima input gambar dan menampilkan hasil prediksi secara langsung.
